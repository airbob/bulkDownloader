Q: I encounter unidentified developer problem when install, how to solve?

A: Method 1: Go to /Applications folder, select bulkDownloader and right click to run

     Method 2:  System Preferences ->Privacy & Security -> General -> unlock -> Allow Application from anywhere

bulkDownloader

bulkDownloader helps you bulk download various types of files from a web page in one go.

 

You can apply certain constrains to the files intend to download and rename them in a batch if needed.
